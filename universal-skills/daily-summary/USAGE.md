# Daily Summary — 使用说明

自动从 Cursor 聊天记录生成每日工作日报，无需外部服务。

## 前置要求

- Python 3.x（仅依赖标准库）
- Cursor 已安装并使用过（需要有聊天历史数据）

## 使用方式

在 Cursor 中直接对 AI 说：

- "帮我生成今天的日报"
- "总结一下昨天的工作"
- "生成 2026-03-08 的工作日报"

AI 会自动读取你的 Cursor 聊天记录，分析工作内容并生成结构化的日报。

## 日报包含内容

- **今日概览**：会话数、涉及项目、代码变更统计
- **工作内容**：按项目分组，每项关联具体会话
- **工作分布**：编码、问题解决、需求讨论、重构、文档、测试等类型占比
- **技术备忘**：待解决事项和后续计划

## 可选功能

- **Git 提交分析**：AI 会询问是否将 Git 提交记录合并到日报中
- **项目过滤**：可指定只分析某个项目的会话
- **历史查询**：可查询之前已保存的日报

## 日报存储

生成的日报会自动保存到本地 SQLite 数据库，可随时查询历史记录。
