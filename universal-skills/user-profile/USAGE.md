# User Profile — 使用说明

分析你的 Cursor 聊天记录，生成个性化用户画像，让 AI 更懂你的编码风格和技术偏好。

## 前置要求

- Python 3.x（仅依赖标准库）
- Cursor 已安装并使用过（需要有聊天历史数据）

## 使用方式

在 Cursor 中直接对 AI 说：

- "帮我生成用户画像"
- "分析一下我的编码习惯"
- "让 Cursor 更了解我"

支持两种范围：
- **项目级**（默认）：只分析当前项目的聊天记录，生成项目相关画像
- **全局**：分析所有项目的聊天记录，生成完整画像

## 画像包含内容

- **编码风格**：命名习惯、架构偏好、注释风格
- **技术画像**：按 Expert / Proficient / Learning 分层的技术栈
- **沟通风格**：提问模式、反馈方式、语言偏好
- **工作习惯**：活跃时段、会话深度、问题解决方式

## 画像如何生效

- 项目级画像自动保存到 `.cursor/rules/user-profile.mdc`，Cursor 会自动加载
- 已自动添加到 `.gitignore`，不会泄露到代码仓库
- 支持增量更新，多次使用会合并已有画像而非完全替换

## 隐私保护

所有数据仅存储在本地，不会上传到任何外部服务。
