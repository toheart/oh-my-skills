# 周报模板

## 输出格式

```markdown
# 周报 - {week_start} ~ {week_end}

## 本周概览
- 工作天数: {working_days} 天
- 总会话数: {total_sessions}
- 涉及项目: {project_count} 个

## 关键成果
- {accomplishment_1}
- {accomplishment_2}
- {accomplishment_3}

## 项目进展

### {project_name_1}
- {work_item_1}
- {work_item_2}

### {project_name_2}
- {work_item_1}

## 工作分布
- 编码: {coding_count}
- 问题解决: {problem_solving_count}
- 需求讨论: {requirements_count}
- 重构: {refactoring_count}
- 文档/测试: {other_count}

## 下周计划
- {plan_item_1}
- {plan_item_2}
```

## 注意事项

- 关键成果最多 5 条，突出最重要的工作
- 按项目分组展示进展
- 下周计划从对话中提取或询问用户
- 使用用户的对话语言撰写
