# Weekly Summary — 使用说明

自动从 Cursor 聊天记录和已保存日报生成每周工作周报，无需外部服务。

## 前置要求

- Python 3.x（仅依赖标准库）
- Cursor 已安装并使用过（需要有聊天历史数据）
- 推荐搭配 daily-summary Skill 使用，已有日报数据时周报质量更高

## 使用方式

在 Cursor 中直接对 AI 说：

- "帮我生成本周的周报"
- "总结一下上周的工作"
- "生成 2026-03-03 ~ 2026-03-09 的周报"

AI 会自动聚合该周的日报数据；如果某些天缺少日报，会回退到直接读取 Cursor 聊天记录。

## 周报包含内容

- **周概览**：工作天数、总会话数、代码变更统计
- **关键成果**：提炼 3-5 条本周重要产出
- **工作详情**：按项目分组展示每日工作
- **工作分布**：编码、问题解决、需求讨论等类型占比
- **下周计划**：基于本周工作自动提取后续事项

## 数据来源

1. 优先读取已保存的日报（由 daily-summary Skill 生成）
2. 缺失日期自动回退到 Cursor 聊天记录

## 周报存储

生成的周报会自动保存到本地 SQLite 数据库，可随时查询历史记录。
